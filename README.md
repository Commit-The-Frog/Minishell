## How to install
1. Clone this repo into your local directory and move into it.
```
git clone https://github.com/K-Minishell/Minishell.git
cd Minishell
```
2. Make minishell
```
make
./minishell
```


## How to remove
```
// removes all object files.
make clean
// removes all object files and also minishell program.
make fclean 
```
